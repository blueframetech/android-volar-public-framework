Class Overview

A VolarVideo media player. It manages the playback of a video from a network stream. Playback occurs in a view called VVPlayerView which is needed in instantiation. You use a VolarPlayerController.Builder to instantiate as shown here: 

 VolarPlayerController player = new VolarPlayerController.Builder(context)
			.setVmapURI(vmapURL)
			.setPlayerView(playerView)
			.load();
 

 Uri uri = getIntent().getData();
 if (uri != null) {
     String vmapURI = uri.toString();

     // Use vmapURI when creating your VolarPlayerController
 }

Consider a VVPlayerView to be an opaque structure. You can add your own custom subviews to layer content on top of the video but you must never modify any of its existing subviews.

This class supports programmatic control of video playback, and user-based control via buttons supplied by the player. This will let you start, stop and seek through content.

To play a network stream whose URL requires access credentials, first connect to your VolarVideo site account by creating a VVCMSAPI instance. With the credential and protection space information in place, you can then play the protected stream.
