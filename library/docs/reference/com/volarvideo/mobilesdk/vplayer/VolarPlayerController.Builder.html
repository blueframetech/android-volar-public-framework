Class Overview

Builder class used to construct a VolarPlayerController. Here's an example usage: 

 VolarPlayerController player = new VolarPlayerController.Builder(context)
		.setVmapURI(vmapURL)
		.setPlayerView(playerView)
		.load();

public VolarPlayerController build ()

Creates a VolarPlayerController with the arguments supplied to this builder. It does not load() the controller. This allows the user to do extra processing before loading the video. Use load() if you don't have any other processing to do and want this to be created and displayed.
