Class Overview

Wrapper for the VolarVideo CMS API. An instance of VVCMSAPIDelegate should be provided to handle responses.

NOTE: Delegate methods are not guaranteed to be called on the same thread they were requested from. Updates to the UI should be done on the UI thread.
